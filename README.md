# ARIA — Augmented Reading Intelligence Assistant
### Powered by Byte Beasts

*Where the page comes alive and every kid belongs in the herd.*

ARIA is an offline-first, AR-powered reading companion for K-12 learners.
A child holds up a Byte Beasts book page, a hand-drawn tech monster
walks off the page in WebAR, and Gemma 4 — running locally via Ollama
— voices the Beast and talks to the child, adapting to their grade level.

Built for rural classrooms with spotty internet. Built on Universal
Design for Learning (UDL) principles. Built to make every kid feel
like they belong.

**Submission for The Gemma 4 Good Hackathon (May 18, 2026).**

---

## The Byte Beasts

Original hand-drawn characters created in Procreate. Each Beast
teaches a STEAM concept through story.

- **Bit** — binary & data — cute + funny
- **Static** — physics & electricity — silly + edgy
- **Lumen** — optics & light — cute + edgy

Two more Beasts live in the broader Byte Beasts universe: **Hex**
(coding & math) and **Cache** (memory & storage).

---

## Tech Stack

- **Gemma 4** via Ollama — local, offline-first inference
- **AR.js + A-Frame** — browser-based WebAR, no app install
- **HTML / CSS / JavaScript** — universal accessibility
- **Python / Node.js bridge** — Ollama API to web client
- **Cloud fallback** — Google Colab / AWS for devices that can't run Gemma locally

---

## Why This Matters

Rural K-12 students often lack reliable broadband, meaning cloud-based
AI tutors fail exactly where they're needed most. ARIA runs on-device,
so a child in a rural holler has the same reading companion as a child
in a wired suburb. The AR layer turns passive reading into an
interactive, embodied experience — and the hand-drawn Byte Beasts give
kids characters they actually want to meet.
